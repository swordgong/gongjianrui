# BMA-Diffusion: 基于多阶段防御的扩散模型概念擦除与语义绕过防御框架

## 项目简介

扩散模型作为一类强大的生成模型，虽然能够生成高质量的数据，但在实际应用中常因滥用而导致侵权问题。现有的概念擦除技术存在一个关键漏洞：**语义绕过**——即当用户使用描述性语句（而非显式概念标签）作为提示词时，模型仍然能生成受限制的概念内容。

为了解决这一问题，本项目提出了 **BMA（Before-Middle-After）多阶段防御框架**。该框架结合了模型微调、语义分析、知识图谱与后处理修正机制，在输入预处理、模型训练和输出后处理三个阶段构建了立体防御体系，显著降低了语义绕过的成功率。

## 主要特点

- **多阶段防御**：覆盖输入、训练、输出三个阶段，不留安全死角
- **精细化概念擦除**：结合闭式优化（CFR）与低秩自适应（LoRA）技术，精准定位并抑制注意力机制中的敏感概念
- **语义关联检测**：集成自然语言处理工具与大语言模型，将描述性输入自动转化为显式概念标签
- **知识图谱增强**：通过构建层次化概念关系图谱，提升模型对复杂语义关联的建模能力
- **自适应内容修正**：基于深度度量学习的后处理评估，自动修正潜在违规生成内容


## 核心方法

### 1. 概念擦除微调

**闭式优化（CFR）**：解析交叉注意力层的投影矩阵闭式解，实现对目标概念的空间精确定位。

**低秩自适应（LoRA）**：利用低秩分解适配器模块，在不改变原始参数的前提下，高效调整注意力机制中 key-value 的映射关系，实现选择性概念抑制。

### 2. 输入预处理防御

通过自然语言处理工具与大型语言模型构建的语义分析模块，自动检测输入提示词是否与受限概念存在语义关联。若存在，则将其转换为显式的概念标签，便于后续擦除模块识别。

### 3. 知识图谱增强建模

构建层次化概念关系的知识图谱，利用实体嵌入技术，显著提升模型对复杂语义关联的建模能力，避免因概念表达方式变化而导致防御失效。

### 4. 后处理自适应修正

采用基于深度度量学习的评估模型，精确计算生成图像与受限概念的视觉相似度。当检测到潜在违规内容时，系统会自适应地触发内容修正机制。

## 📊 实验结果

本方案在标准测试集上进行了评估，实验结果表明：

| 指标 | 基线模型 | BMA 框架 |
| :--- | :---: | :---: |
| 语义绕过成功率 | 66% | **< 10%** |

相比现有概念擦除方法，BMA 框架在防御语义绕过攻击方面取得了显著效果提升。
